## Component Library (static)

A modern, developer-focused component library site where users can:

- Browse reusable UI components
- See a live preview
- Copy a syntax-highlighted code snippet
- Filter/search by name, description, slug, or tags

Built with **Next.js + Tailwind CSS + shadcn-style UI primitives**, and configured for **fully static export**.

## Getting Started

Install deps and run the dev server:

```bash
npm install
npm run dev
```

Build a static export:

```bash
npm run build
```

The static output is generated by Next.js export mode (`output: "export"` in `next.config.mjs`).

## Adding components

Components live in `src/components/library/<slug>/`:

- `meta.ts`: name, slug, description, tags
- `preview.tsx`: live preview UI (rendered on cards and detail page)
- `source.tsx`: the actual component implementation
- `code.ts`: the copyable code string shown on the docs page

Register new components in `src/lib/component-registry.ts`.

## Routes

- **Home**: `src/app/page.tsx`
- **Component page**: `src/app/components/[slug]/page.tsx` (static params generated from the registry)
